Part 2

Here's an example of code using functional style iteration with reduce and filter:

    var emps = [
                {id: 1, name: 'J. Jones', salary: 94000.00},
                {id: 2, name: 'A. Smith', salary: 54000.00},
                {id: 3, name: 'B. Ju', salary: 62000.00},
                {id: 4, name: 'A. C. Linn', salary: 75000.00}
            ]
            
            // first calculate the total salary
            var totalSalary = emps.reduce( function(prev, cur) {
                return prev = cur.salary;
            }, 0); 
            
    
            // next calculate the average salary
            var averageSalary = totalSalary / emps.length;
    
            // here highestPaid will hold an array of all employees whose salary is greater than the calculated average.
            var highestPaid = emps.filter(function (emp){ 
                 return emp.salary >=  averageSalary;
             })

Be sure to review and understand the code. Copy and paste it into an editor and implement the following:

  1. Find the employee whose id is 1;

  2. Use map and JSON.stringify() to return an array of objects ready for output.

  3. Use a forEach iteration to display the results of #2 to the console.

  4. Implement a solution to display to the console the name and salary of each employee whose salary is greater than 70000, sorted in descending order of salary.

  5. Here's a tricky one: implement a way to answer the question 'Are there any employees whose salary is greater than 70000?'.
